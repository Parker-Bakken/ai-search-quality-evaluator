# Query Evaluation Guidelines

This document outlines how search results should be evaluated for relevance.

## Evaluation Criteria

### 1. Relevance
Does the search result directly relate to the user's query?

Score range:
0.0 = unrelated  
0.5 = partially relevant  
1.0 = highly relevant

### 2. Intent Match
Does the result satisfy the user's likely intent?

Example intents:
- informational
- navigational
- transactional

### 3. Informational Usefulness
Does the result provide helpful and accurate information?

## Example

Query:
best productivity apps for students

Result:
Top productivity apps every college student should use

Evaluation:
Relevance: 1.0  
Intent Match: 1.0  
Usefulness: 0.9

## Purpose

These guidelines simulate the process used by human evaluators in search engine quality rating tasks.
